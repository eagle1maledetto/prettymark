# MSIX Visual Assets

Place the following PNG files here (transparent background, square):

| File                    | Size      | Used for                         |
|-------------------------|-----------|----------------------------------|
| `StoreLogo.png`         | 50x50 px  | Store listing icon               |
| `Square44x44Logo.png`   | 44x44 px  | Taskbar, Start menu (small)      |
| `Square150x150Logo.png` | 150x150 px| Start menu tile                  |

Use your app logo centered on a transparent background.
These are required by `build-msix.ps1` to create the MSIX package.
